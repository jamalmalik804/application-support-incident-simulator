# Application Support Incident Simulator

## Overview
This project simulates real-world application support scenarios using SQL, log analysis, and structured troubleshooting.

## Skills Demonstrated
- Application Support & Incident Handling
- SQL Debugging & Data Validation
- Log Analysis
- Root Cause Analysis
- ITIL Concepts (Incident & Problem Management)

## Technologies
- SQL (MS SQL Server / MySQL)
- Windows / Linux (basic)
- Manual log analysis

## Project Structure
- database/ → schema & sample data
- incidents/ → real-world issue scenarios
- queries/ → debugging SQL queries
- logs/ → simulated application logs
- screenshots/ → proof of execution
